01: First HTML page

Let's start with a simple example of HTML code. Choose your preferred text editor and enter the code below: (Note: Type, don't copy! That builds connections in your brain to help your learning process.)

    <html>
      <body>
        Hello, funny world! This is my first HTML page!
      </body>
    </html>

Save the file with the name index.html and open it in your favorite browser (e.g., Chrome, Firefox, Safari, Internet Explorer). The result will look like this:

Now let's increment this code and analyze it. (Update and review your file in the browser.)

  <!DOCTYPE html>
  <html>
    <!-- This is the sintax of a comment -->
    <head>
      <title> My first HTML page! </title>
    </head>
    <body>
      Hello, funny world! This is my first HTML page!
    </body>
  </html>
  • HTML works with what we call tags. The syntax of an HTML tag uses the symbols <>.
  • Some tags need to be declared in pairs. They open and close the scope of the tag, for example <html> </html>. We will see examples of tags with a single declaration, which do not need to be closed.
  • HTML pages should begin with the <!DOCTYPE html> statement to help your browser identify the type of document it will interpret. Your browser may not interpret some tags well if the instruction is not declared.
  • The entire page structure is between the <html> </html> tags.
  • The initial structure of the HTML document contains what we call a header. The <head> </head> tags delimit the header and can contain information about the page. The sample code adds a title to our page, which will appear in the top bar of your browser.
  • After the header, the <body> </body> tags delimit the page's scope. The "body" will contain all the content available to the user, such as texts, videos, and images.
  • Comments are information not exhibited by the browser. They are useful for documenting your code. Any text between <!-- and --> is a comment.

Let's continue improving our code and getting acquainted with more HTML tags. We're going to explore then during the class:
