AULA 01: Primeira página em HTML

Vamos começar com um exemplo simples de código HTML. Escolha um editor de texto de sua preferência e digite o código abaixo: (Obs: Digite, não copie! Isso ajuda no seu aprendizado e na criação de conexões no seu cérebro! )

    <html>
    <body>
      Olá mundo!
      Esta é minha primeira página em HTML!
    </body>
    </html>

Agora salve o arquivo com nome: index.html e abra o arquivo no seu browser favorito (Chrome, Firefox, Safari, Internet Explorer, ...). O resultado será:

Agora vamos incrementar esse código (Visualize novamente o arquivo no browser!), e analizar as suas informações:

  <!DOCTYPE html>
  <html>
    <head>
      <title> Minha página HTML </title>
    </head>
    <body>
      Olá mundo!
      Esta é minha primeira página em HTML!
    </body>
  </html>
  • HTML trabalha com o que chamamos de tags. A sintaxe de uma tag HTML utiliza os símbolos <>.
  • Algumas tags precisam ser declaradas em pares, que abrem e fecham o escopo da tag, por exemplo <html> </html>. Veremos exemplos de tags que possuem declaração única, não precisam ser fechadas.
  • Todos as páginas HTML devem começar com a declaração <!DOCTYPE html> para auxiliar o seu browser a identificar o tipo do documento que ele interpretará.
  • Toda a estrutura da página está contida entre as tags <html> </html>
  • A estrutura inicial do documento HTML contém o que chamamos de cabeçalho. A tag <head> </head> delimita o cabeçalho, e pode conter informações sobre a página. No exemplo acima adicionamos um título à nossa página, que aparecerá na barra superior do seu browser.
